# 기계진동 강의 노트 정리

## 1. 기본 개념

### 진동의 정의
- **진동(Vibration)**: 주기 T로 반복되는 운동

### 에너지 요소
| 요소 | 에너지 종류 | 기호 |
|------|------------|------|
| 질량 (mass) | 운동 에너지 (Kinetic Energy) | $m$ |
| 스프링 (spring) | 위치 에너지 (Potential Energy) | $k$ |
| 댐퍼 (damper) | 에너지 손실 (Loss of Energy) | $c$ |

---

## 2. 운동 방정식 (EOM)

기본 운동방정식:

$$m\ddot{x} + c\dot{x} + kx = F(t)$$

### 케이스별 분류
1. **비감쇠 자유진동**: $m\ddot{x} + kx = 0$ → 자유진동
2. **감쇠 자유진동**: $m\ddot{x} + c\dot{x} + kx = 0$
3. **비감쇠 강제진동**: $m\ddot{x} + kx = F(t)$ (외력이 harmonic인 경우 포함)
4. **감쇠 강제진동**: $m\ddot{x} + c\dot{x} + kx = F(t)$

**✔ General Solution = Homogeneous + Particular**

---

## 3. 진동의 분류

### 자유도(DOF)에 따른 분류
- **무한 자유도**: 연속(continuous) 또는 분포된(distributed) 시스템
- **유한 자유도**: 이산(discrete) 시스템

### 자유진동 vs 강제진동
- **자유진동**: 외력 = 0 → $\omega_n = \sqrt{k/m}$
- **강제진동**: 외력 ≠ 0

### 선형 vs 비선형
- **선형(linear)**: 모든 구성요소가 선형 → 선형 미분방정식
- **비선형(nonlinear)**: 하나의 구성요소라도 비선형 → 비선형 방정식

### 결정론적 vs 비결정론적
- **Deterministic**: 작용하는 힘 또는 운동이 t에 대해 알려져 있음
- **Non-deterministic**: 예) 바람, 지진 등 불규칙 입력

---

## 4. 단자유도(Single DOF) 비감쇠 자유진동

### 운동방정식 유도

**운동방정식:**
$$m\ddot{x} + kx = 0$$

정적 처짐 $\delta_{st}$를 이용하면:
$$mg = k\delta_{st}$$

에너지 보존 방법으로도 동일하게 유도됨:
$$m\dot{x}\ddot{x} + (k\delta_{st} - mg)\dot{x} + kx\dot{x} = 0 \implies m\ddot{x} + kx = 0$$

### 고유진동수 및 주기

$$\omega_n = \sqrt{\frac{k}{m}}, \quad f = \frac{\omega_n}{2\pi} = \frac{1}{2\pi}\sqrt{\frac{k}{m}}, \quad T = \frac{1}{f} = 2\pi\sqrt{\frac{m}{k}}$$

정적 처짐과의 관계:
$$mg = k\delta_{st} \implies \frac{m}{k} = \frac{\delta_{st}}{g} \implies T = 2\pi\sqrt{\frac{\delta_{st}}{g}}$$

### 초기조건을 이용한 해

**해의 형태 (sine):**
$$x(t) = A\sin(\omega_n t + \phi)$$

초기조건 $x(0) = x_0,\ \dot{x}(0) = v_0$ 대입:

$$\tan\phi = \frac{x_0 \omega_n}{v_0}, \quad A = \sqrt{x_0^2 + \frac{v_0^2}{\omega_n^2}}$$

$$\boxed{x(t) = \sqrt{x_0^2 + \frac{v_0^2}{\omega_n^2}} \cdot \sin\!\left(\sqrt{\frac{k}{m}}\,t + \tan^{-1}\!\left(\frac{x_0\omega_n}{v_0}\right)\right)}$$

**해의 형태 (cosine):**
$$x(t) = x_0\cos\omega_n t + \frac{v_0}{\omega_n}\sin\omega_n t$$

## 5. 등가 강성(Equivalent Stiffness)

### 직렬 연결 (series)

$$\frac{1}{k_{eq}} = \frac{1}{k_1} + \frac{1}{k_2}$$

### 병렬 연결 (parallel)

$$k_{eq} = k_1 + k_2$$

### 구조 요소의 강성

**외팔보 (cantilever beam):**
$$\delta_{st} = \frac{Wl^3}{3EI} \implies k = \frac{3EI}{l^3}$$

**단면 2차 모멘트:**
$$I_{beam} = \frac{1}{12}at^3$$

**축(rod)의 축방향 강성:**
$$k_{rod} = \frac{AE}{l} = \frac{\pi d^2 E}{4l}$$

### 비틀림 강성 (torsional stiffness)

$$M_t = \frac{GJ}{l}\theta, \quad k_t = \frac{GJ}{l}$$

- $G$: 전단 탄성계수(shear modulus)
- $J$: 극관성 모멘트(polar moment of inertia)
- 원형 단면: $J_{극} = \frac{\pi d^4}{32}$

---

## 6. 등가 질량(Equivalent Mass)

에너지 동치 조건 $T = \frac{1}{2}m_{eq}\dot{x}^2$을 이용:

**도르래-질량 시스템:**
$$T_{eq} = \frac{1}{2}m_{eq}\dot{x}^2 = \frac{1}{2}m\dot{x}^2 + \frac{1}{2}J_0\left(\frac{\dot{x}}{r}\right)^2 \implies m_{eq} = m + \frac{J_0}{r^2}$$

**구름 접촉 포함:**
$$m_{eq} = m + \frac{J_b}{l_1^2} + \frac{J_s}{5}\frac{l_2^2}{l_1^2} + m_s\frac{l_2^2}{l_1^2}$$

---

## 7. 단진자 및 진자형 시스템

### 단순 진자 (simple pendulum)

운동방정식 ($\sin\theta \approx \theta$ 선형화):
$$J_0\ddot{\theta} + mgd\,\theta = 0 \implies \omega_n = \sqrt{\frac{mgd}{J_0}}$$

**등가 길이:**
$$l = \frac{J_0}{md}$$

### 복합 진자 (compound pendulum)

**평행축 정리:** $J_0 = J_G + md^2$, 즉 $mk_0^2 = mk_G^2 + md^2$

**등가 길이:** $l = \frac{k_G^2}{d} + d$

**타점 중심(center of percussion):** $\gamma = l - d = \frac{k_G^2}{d}$

---

## 8. Simple Harmonic Motion (SHM)

$$s = s_0\sin\omega t, \quad v = s_0\omega\cos\omega t, \quad a = -s_0\omega^2\sin\omega t = -s\omega^2$$

---

## 9. 복소수 표현 및 맥놀이(Beat)

### 오일러 공식

$$\vec{r}_A = Re^{j\theta} = R\cos\theta + jR\sin\theta$$

$$\cos\theta = \frac{e^{j\theta}+e^{-j\theta}}{2}, \quad \sin\theta = \frac{e^{j\theta}-e^{-j\theta}}{2i}$$

### 맥놀이 (Beat)

두 신호의 합:
$$x_1(t) = A\cos\omega t, \quad x_2(t) = A\cos(\omega + \delta)t$$

$$x_1 + x_2 = 2A\cos\frac{\delta}{2}t\cdot\cos\!\left(\omega + \frac{\delta}{2}\right)t$$

**Beat Period** $= \frac{2\pi}{\delta}$

---

## 10. 푸리에 급수 (Fourier Series)

### 삼각함수 형태

$$y(t) = \frac{A_0}{2} + \sum_{n=1}^{\infty}\left[A_n\cos(n\omega t) + B_n\sin(n\omega t)\right]$$

$$A_0 = \frac{\omega}{\pi}\int_0^{2\pi/\omega}y(t)\,dt, \quad A_n = \frac{\omega}{\pi}\int_0^{2\pi/\omega}y(t)\cos(n\omega t)\,dt$$

$$B_n = \frac{\omega}{\pi}\int_0^{2\pi/\omega}y(t)\sin(n\omega t)\,dt$$

### 복소 지수 형태

$$y(t) = \sum_{n=-\infty}^{\infty} C_n e^{jn\omega t}$$

$$C_n = \frac{a_n}{2} - \frac{jb_n}{2}, \quad C_{-n} = \frac{a_n}{2} + \frac{jb_n}{2}$$

---

## 11. 감쇠 시스템 (Damped System)

### 감쇠 자유진동

$$m\ddot{x} + c\dot{x} + kx = 0$$

특성방정식: $-m\omega^2 + k = 0 \implies \omega_n = \sqrt{k/m}$

**감쇠 고유진동수:**
$$\omega_d = \sqrt{1-\zeta^2}\,\omega_n$$

### 고유진동수의 물리적 의미
- 고유진동수(natural frequency) $\omega_n$: 탄성력과 관성력이 균형을 이루는 진동수
- $\omega < \omega_n$: quasi-static (강성 지배)
- $\omega > \omega_n$: inertia dominance (관성 지배)
- $\omega = \omega_n$: **공진(resonance)**

## 12. 등가 강성 – 복합 예제

### 예제 1: 외팔보 + 로드 조합

$$k_{beam} = \frac{Eat^3}{4b^3}, \quad k_{rod} = \frac{\pi d^2 E}{4l}$$

$$\frac{1}{k_{eq}} = \frac{1}{k_b} + \frac{1}{k_r} = \frac{4b^3}{Eat^3} + \frac{4l}{E\pi d^2}$$

$$k_{eq} = \frac{E}{4}\left(\frac{\pi d^2 b^3}{\pi d^2 b^3 + lat^3}\right) \cdot \frac{1}{\text{(분모 항)}}$$

### 예제 2: 레버 위의 스프링 조합

$$x = \frac{l_2 x_1 + l_1 x_2}{l_1 + l_2}$$

모멘트 평형 및 $F = kx$ 적용:

$$k_{eq} = \frac{k_1 k_2 (l_1 + l_2)^2}{k_1 l_1^2 + k_2 l_2^2}$$

### 예제 3: 경사면 위 질량 + 두 스프링 ($c<0$)

정적 처짐 관계:
$$\delta_{st}(k_1 + k_2) = mg\sin\theta$$

운동방정식:
$$m\ddot{x} + (k_1 + k_2)x = 0 \implies \omega_n = \sqrt{\frac{k_1 + k_2}{m}}$$

### 예제 4: 대칭 스프링 배열 ($k, 4k$)

$$x = 2p/k + 2p/k = 4p/k$$

$$k_{eq} = \frac{p}{x} = \frac{p}{4p/k} = \frac{k}{4}, \quad \omega_n = \frac{1}{2}\sqrt{\frac{k}{m}}$$

## 13. 수치 예제

### 예제 5
$m=5\,\text{kg},\ k=2000\,\text{N/m},\ x_0=20\times10^{-3}\,\text{m},\ v_0=200\times10^{-3}\,\text{m/s}$

$$\omega_n = \sqrt{\frac{k}{m}} = \sqrt{400} = 20\,\text{rad/s}$$

$$x(t) = 0.02\cos 20t + 0.01\sin 20t = 0.01\sqrt{5}\cos(20t - \tan^{-1}\tfrac{1}{2})$$

### 예제 6
$m=2\,\text{kg},\ k=3200\,\text{N/m},\ x_0=0,\ v_0=50.1\,\text{m/s}$

$$\omega_n = 40\,\text{rad/s} \implies x(t) = \frac{v_0}{40}\sin(40t)$$

최대속도: $\dot{x}_{max} = v_0 = 50.1 \approx 54\,\text{m/s}$

## 14. EX3 – 복합 진자 예제

### 예제 ①: 균일 막대 + 끝단 질량

$$I_0\ddot{\theta} + k_G\theta = 0, \quad I_0 = Ml^2 + \frac{1}{3}ml^2 = \left(M+\frac{m}{3}\right)l^2$$

$$k_G = \left(M+\frac{m}{2}\right)gl \implies \omega_n = \sqrt{\frac{\left(M+\frac{1}{2}m\right)g}{\left(M+\frac{1}{3}m\right)l}}$$

### 예제 ②: 두 개의 샤프트 직렬 연결

$$k_i = \frac{G J_{p,i}}{l_i}, \quad J_{p,i} = \frac{\pi d_i^4}{32}$$

$$k_1 = \frac{G}{l_1}\cdot\frac{\pi d_1^4}{32}, \quad k_2 = \frac{G\pi d_2^4}{l_2 \cdot 32}$$

직렬 등가 강성:
$$k_t = \frac{\pi G}{32}\cdot\frac{1}{\dfrac{l_1}{d_1^4}+\dfrac{l_2}{d_2^4}}$$

### 예제 ③: 진자 초기조건

$\omega_0 = 1 = \dot{\theta}_0,\ \theta(0)=0,\ A=0.5$

$$\theta(t) = A\sin\omega_n t, \quad \dot{\theta}(t) = A\omega_n\cos\omega_n t$$

$$1 = A\omega_n \implies \omega_n = 2, \quad l = \frac{g}{\omega_n^2} = \frac{10}{4} = 2.5\,\text{m}$$

### 예제 ④: 내부 편심 원판 + 두 스프링

$x = 2R\theta$ (구름 접촉 조건, 슬립 없음)

$$k_{eq} = (k_1+k_2)\left(1+\frac{a}{R}\right)^2, \quad m_{eq} = m + \frac{J}{R^2}$$

$$\omega_n = \left(1+\frac{a}{R}\right)\sqrt{\frac{k_1+k_2}{m+J/R^2}}$$

## 15. 불안정 시스템 분석 (EX – 막대 + 스프링)

$$T = \frac{1}{2}J\dot{\theta}^2, \quad U = 2\cdot\frac{1}{2}k(l\sin\theta)^2 + mgd\cos\theta$$

선형화 ($\sin\theta \approx \theta$):

$$J\ddot{\theta} + \left(2kl^2 - \frac{mgl}{2}\right)\theta = 0$$

$$\beta = \frac{12kl^2 - 3mgl}{2ml^2}$$

**안정성 판단:**
1. $\beta > 0$: 안정(stable) → $\theta(t) = A_1\cos\omega_n t + A_2\sin\omega_n t$
2. $\beta = 0$: 등속 운동 → $\theta(t) = C_1 t + C_2$
3. $\beta < 0$: **불안정(unstable)** → $\theta(t) = B_1 e^{\alpha t} + B_2 e^{-\alpha t}$
