# Ch6. 보존 방정식 — 오버뷰

## 핵심 질문

> 유체 요소 하나에 물리 법칙을 적용하면 어떤 방정식이 나오는가?

---

## 전체 흐름

```
미소 요소의 물리량 변화 표현
        ↓
① 질량 보존  →  연속 방정식 (∇·V = 0)
        ↓
② 운동량 보존  →  응력 텐서 정의
        ↓
   구성 방정식 (Newtonian fluid 가정)
        ↓
   나비에-스토크스 방정식
        ↓
③ 에너지 보존  →  온도장 방정식
```

---

## 각 파트 한 줄 요약

| 파트 | 내용 | 결과 방정식 |
|------|------|------------|
| 미소 변화 | 테일러 전개로 고차항 무시 | $f(x+\delta x) \approx f(x) + f'\delta x$ |
| 질량 보존 | 검사체적 출입 질량 = 내부 변화 | $\nabla\cdot\vec{V}=0$ |
| 물질 미분 | 입자를 따라가는 변화율 | $\frac{D}{Dt} = \frac{\partial}{\partial t} + (\vec{V}\cdot\nabla)$ |
| 운동량 보존 | $F=ma$를 유체 요소에 적용 | 응력 텐서 포함 운동 방정식 |
| 구성 방정식 | 응력 ↔ 변형률 관계 (뉴턴 유체) | $\tau_{ij} = \mu(\partial_i u_j + \partial_j u_i)$ |
| N-S 방정식 | 운동량 + 구성 방정식 결합 | $\rho\frac{D\vec{V}}{Dt} = -\nabla p + \mu\nabla^2\vec{V} + \rho\vec{g}$ |
| 에너지 보존 | 열 + 일 = 내부에너지 변화 | $\rho C\frac{DT}{Dt} = k\nabla^2 T + \mu\Phi$ |

---

## N-S 방정식의 난이도와 해석 전략

N-S 방정식은 **비선형 PDE** → 일반 해석해 없음

레이놀즈 수 $Re = \frac{\rho VL}{\mu}$ 로 두 항의 크기를 비교:

$$Re \ll 1 \Rightarrow \text{점성 지배, 선형화 가능}$$
$$Re \gg 1 \Rightarrow \text{관성 지배, 포텐셜 유동 근사}$$

---

## §6.4–6.6 포텐셜 유동으로의 확장

비점성($\mu \to 0$) + 비압축성 + 비회전($\nabla\times\vec{V}=0$) 가정 시:

$$\nabla^2\phi = 0 \quad \text{(라플라스 방정식)}$$

**선형 방정식** → 기본 해를 중첩(superposition)하여 복잡한 유동 구성 가능

| 기본 해 | 포텐셜 $\phi$ | 특징 |
|---------|-------------|------|
| 균일 유동 | $Ur\cos\theta$ | 일정 속도 |
| 소스/싱크 | $\frac{m}{2\pi}\ln r$ | 방사형 유동 |
| 와류 | $\frac{\Gamma}{2\pi}\theta$ | 원형 유동 |
| 이중극 | $\frac{k\cos\theta}{r}$ | 소스+싱크 극한 |

→ 조합 예시: **원형 실린더** (균일유동 + 이중극), **회전 실린더** (+ 와류) → 마그누스 효과

# Ch7. 차원 해석, 상사성 및 모델링 — 오버뷰

## 핵심 질문

> 실험 변수가 너무 많을 때, 어떻게 효율적으로 줄일 수 있는가?

---

## 전체 흐름

```
물리 변수 k개 나열
        ↓
기본 차원 r개 파악 (M, L, T 또는 F, L, T)
        ↓
버킹엄 파이 정리 적용
        ↓
(k - r)개의 무차원 Π 그룹 도출
        ↓
실험 or 해석으로 Π들 사이 관계 결정
        ↓
모델-실물 상사성 조건 수립 → 스케일 예측
```

---

## 왜 필요한가?

파이프 압력 강하 예시:

$$\Delta P_\ell = f(D,\, \rho,\, \mu,\, V)$$

각 변수별로 실험하면 그래프가 **4개** 필요.  
차원 해석 후:

$$\frac{D\Delta P_\ell}{\rho V^2} = \phi\left(\frac{\mu}{\rho VD}\right)$$

→ 그래프 **1개**로 압축. 실험 횟수 대폭 감소.

---

## 버킹엄 파이 정리 핵심

$$\text{변수 } k\text{개},\quad \text{기본 차원 } r\text{개} \quad\Rightarrow\quad (k-r)\text{개의 무차원 } \Pi \text{ 그룹}$$

**반복 변수 선택 기준:** 기본 차원을 모두 포함하는 $r$개의 변수 선택  
(서로 독립적이어야 하며, 구하려는 변수는 제외)

---

## 각 파트 한 줄 요약

| 파트 | 내용 |
|------|------|
| 파이 정리 | 변수 $k$개 → $(k-r)$개 무차원 그룹으로 축약 |
| 예제: 진자 | $\omega = f(\ell, m, g)$ → $\omega = C\sqrt{g/\ell}$ 도출 |
| 예제: 파이프 | $\Delta P_\ell = f(D,\rho,\mu,V)$ → $\Pi_1 = \phi(\Pi_2)$ 즉 $f(Re)$ |
| 예제: 항력 | 모양·Re에 따른 무차원 항력계수 도출 |
| 상사성 | 모델과 실물의 $\Pi$ 그룹 일치 → 힘 스케일링 |

---

## 상사성(Similitude)의 의미

모델 실험에서 **같은 무차원 그룹값**을 만족시키면,  
실물의 거동을 모델로 예측할 수 있다.

$$\Pi_{i,\,\text{모델}} = \Pi_{i,\,\text{실물}} \quad \forall\, i$$

$$\Rightarrow F_D = \left(\frac{w}{w_m}\right)^2 \cdot \frac{\rho}{\rho_m} \cdot \left(\frac{V}{V_m}\right)^2 \cdot F_{D,m}$$

> **핵심:** 차원 해석은 물리 방정식을 몰라도 쓸 수 있다.  
> 단, 어떤 변수가 관련되는지는 **물리적 통찰**이 필요하다.


# Ch8. 파이프 내 점성 유동 — 오버뷰

## 핵심 질문

> 실제 파이프를 흐르는 유체에서 속도 분포와 압력 손실은 어떻게 결정되는가?

---

## 전체 흐름

```
층류 vs 난류 판별 (Re 기준)
        ↓
입구 영역 → 완전 발달 영역
        ↓
┌─────────────────┬──────────────────┐
│   완전 발달 층류  │   완전 발달 난류   │
│  (이론 해석 가능) │  (경험적 모델 필요) │
└─────────────────┴──────────────────┘
        ↓
에너지 방정식으로 압력 손실 정량화
        ↓
주손실 (Darcy-Weisbach) + 부손실
```

---

## 각 파트 한 줄 요약

| 파트 | 내용 | 핵심 결과 |
|------|------|----------|
| §8.1 일반 특성 | Re로 유동 형태 분류, 입구 길이 | $Re<2000$ 층류, $Re>4000$ 난류 |
| §8.2 완전 발달 층류 | N-S 방정식의 드문 완전 해석해 | 포물선 속도 분포, 포아죄유 법칙 |
| §8.3 완전 발달 난류 | 시간 평균 + 레이놀즈 응력 | 평탄한 속도 분포, 멱함수 법칙 |
| §8.4 파이프 손실 | 에너지 방정식 + 마찰계수 | 다르시-바이스바흐, 무디 선도 |

---

## §8.2 핵심: 층류의 완전 해석

세 가지 방법(뉴턴 제2법칙 / N-S 방정식 / 차원 해석) 모두 같은 결론:

$$u(r) = \frac{\Delta p D^2}{16\mu\ell}\left[1-\left(\frac{2r}{D}\right)^2\right] \quad \text{(포물선)}$$

$$Q = \frac{\pi D^4}{128\mu\ell}\Delta p \quad \text{(포아죄유 법칙)}$$

$$f = \frac{64}{Re} \quad \text{(층류 마찰계수)}$$

---

## §8.3 핵심: 난류의 복잡성

층류와 결정적으로 다른 점:

$$\tau_{\text{총}} = \underbrace{\mu\frac{d\bar{u}}{dy}}_{\text{층류 기여}} + \underbrace{(-\rho\overline{u'v'})}_{\text{레이놀즈 응력}}$$

- 레이놀즈 응력은 **와류에 의한 운동량 교환**으로 발생
- 닫힌 형태의 해가 없음 → 경험적 모델 필요 (멱함수 법칙, 혼합 길이 모델)

---

## §8.4 핵심: 압력 손실 계산

$$h_L = f \cdot \frac{\ell}{D} \cdot \frac{V^2}{2g}$$

| 변수                       | 결정 방법                   |
| ------------------------ | ----------------------- |
| 마찰계수 $f$                 | 층류: $64/Re$ / 난류: 무디 선도 |
| 조도 $\varepsilon/D$       | 파이프 재질에 따라 주어짐          |
| 부손실 $h_{L,\text{minor}}$ | 각 부품의 손실계수 $K$ 이용       |

> **핵심:** §8.2는 유체역학에서 **큰 가정 없이** 완전히 풀리는 몇 안 되는 문제 중 하나.  
> 난류는 아직도 완전히 해결되지 않은 문제이며, 실용적으로는 실험 상관식에 의존한다.
