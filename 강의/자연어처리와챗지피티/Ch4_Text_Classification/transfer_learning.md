# 전이학습 (Transfer Learning)

#분류 #전이학습 #다국어

## 개념

대규모 데이터로 사전 학습(pre-training)된 모델을 특정 태스크에 맞게 미세 조정(fine-tuning).

```
Pre-training (대규모 일반 데이터)
        ↓
   사전 학습 모델
        ↓
Fine-tuning (소규모 태스크 특화 데이터)
        ↓
   태스크 특화 모델
```

## Cross-lingual Learning (교차 언어 학습)

다국어 간 전이학습. 리소스가 풍부한 언어(예: 영어)에서 학습 후 다른 언어에 적용.

**XNLI (Cross-lingual NLI):**
- MNLI를 15개 언어로 번역
- 인간 번역 + 기계 번역 포함

### 시나리오

| 시나리오 | 방법 |
|----------|------|
| A: 오프라인 기계 번역 | 입력 텍스트를 영어로 번역 후 영어 모델 적용 |
| B: Cross-lingual 모델 | 다국어 모델을 직접 한국어에 적용 |

## Multi-task Learning

여러 태스크를 동시에 학습 → 태스크 간 공유 표현 학습.

**예시: 영화 대본 위험 등급 예측**
- 연령 제한, 폭력성, 선정성 등 여러 관점 동시 분류
- 공유 인코더 + 태스크별 분류 헤드
