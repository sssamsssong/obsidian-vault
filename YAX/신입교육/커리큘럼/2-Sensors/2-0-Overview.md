# Phase 2: 센서 통합 개요

> YAX F1TENTH 교육자료 | Phase 2: Sensors
> 
> 기간: 7-10주차

---

## 1. Phase 2 소개

### 학습 목표
1. Hokuyo UST-10LX LiDAR를 ROS2와 연동할 수 있다
2. LiDAR 데이터를 처리하고 분석할 수 있다
3. Intel RealSense D435 카메라를 설정할 수 있다
4. RViz2로 센서 데이터를 시각화할 수 있다
5. 다중 센서 데이터를 동기화할 수 있다

### 전체 소요 시간
- 오프라인 세션: 8시간 (4주 × 2시간)
- 자율학습: 8시간
- **총 16시간**

### 왜 센서 통합이 중요한가?
자율주행의 핵심은 주변 환경을 정확하게 인식하는 것입니다. LiDAR와 카메라는 F1TENTH 차량의 "눈" 역할을 합니다.

## 2. 센서 구성

### 2.1 Hokuyo UST-10LX LiDAR

| 항목 | 사양 |
|------|------|
| 타입 | 2D LiDAR |
| 범위 | 0.06m ~ 10m |
| 각도 범위 | 270° |
| 각도 해상도 | 0.25° |
| 스캔 주파수 | 40Hz |
| 인터페이스 | USB / Ethernet |

**용도**: 장애물 감지, 벽 따라가기, Gap 감지

### 2.2 Intel RealSense D435

| 항목 | 사양 |
|------|------|
| 타입 | RGB-D 카메라 |
| RGB 해상도 | 1920×1080 @ 30fps |
| Depth 해상도 | 1280×720 @ 90fps |
| Depth 범위 | 0.1m ~ 10m |
| FOV | 87° × 58° |
| 인터페이스 | USB 3.0 |

**용도**: 시각 인식, 뎁스 기반 장애물 감지, 데이터 수집

### 2.3 센서 배치

```
        [RealSense D435]
              │
    ┌─────────┴─────────┐
    │                   │
    │    F1TENTH 차량    │
    │                   │
    └─────────┬─────────┘
              │
       [UST-10LX LiDAR]
```

## 5. 필수 패키지 설치

```bash
# LiDAR 드라이버
sudo apt install ros-humble-urg-node

# RealSense 드라이버
sudo apt install ros-humble-realsense2-camera
sudo apt install ros-humble-realsense2-description

# 시각화 도구
sudo apt install ros-humble-rviz2

# 이미지 처리
sudo apt install ros-humble-cv-bridge
sudo apt install ros-humble-image-transport
```
