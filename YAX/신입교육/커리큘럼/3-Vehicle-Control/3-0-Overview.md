# Phase 3: 차량 제어 개요

> YAX F1TENTH 교육자료 | Phase 3: Vehicle Control
> 
> 기간: 11-14주차

---

## 1. Phase 3 소개

### 학습 목표
1. VESC 모터 컨트롤러를 설정하고 튜닝할 수 있다
2. ROS2 드라이버로 차량을 제어할 수 있다
3. Ackermann 조향 원리를 이해할 수 있다
4. 조이스틱으로 차량을 수동 조종할 수 있다
5. 오도메트리를 캘리브레이션할 수 있다

### 전체 소요 시간
- 오프라인 세션: 8시간 (4주 × 2시간)
- 자율학습: 8시간
- **총 16시간**

### 왜 차량 제어가 중요한가?
자율주행의 최종 출력은 차량 제어입니다. 센서와 알고리즘이 아무리 뛰어나도 정확한 제어가 없으면 의미가 없습니다.

## 2. 하드웨어 구성

### 2.1 VESC 6+

| 항목 | 사양 |
|------|------|
| 입력 전압 | 8-60V (3S-12S LiPo) |
| 연속 전류 | 50A |
| 최대 전류 | 150A |
| 인터페이스 | USB, CAN, PPM, UART |
| 특징 | FOC 제어, 센서리스 |

**역할**: BLDC 모터 구동, 서보 신호 출력

### 2.2 연결 구성

```
┌─────────────────────────────────────────────────────────────┐
│                     F1TENTH 차량                            │
├─────────────────────────────────────────────────────────────┤
│                                                             │
│   [Jetson Orin Nano]                                        │
│        │ USB                                                │
│        ▼                                                    │
│   [VESC 6+] ──► BLDC Motor (후륜 구동)                       │
│        │                                                    │
│        └──► Servo (전륜 조향)                                │
│                                                             │
│   [Battery 3S LiPo] ──► VESC 전원                           │
│                                                             │
└─────────────────────────────────────────────────────────────┘
```

### 2.3 Ackermann 조향

```
         ┌───────────────────┐
         │                   │
    [FL] │                   │ [FR]
    ◄────┤    차량 바디      ├────►
         │                   │
         │        *          │  ← 회전 중심
         │                   │
    [RL] │                   │ [RR]
    ●────┤                   ├────●
         └───────────────────┘
         
    FL/FR: 전륜 (조향)
    RL/RR: 후륜 (구동)
```

## 5. 안전 주의사항

### ⚠️ 필수 안전 수칙

1. **배터리 관리**
   - 충전 시 화재 감시
   - 과방전 방지 (3.3V/셀 이하 금지)
   - 손상된 배터리 사용 금지

2. **차량 테스트**
   - 첫 테스트는 바퀴를 들어올린 상태에서
   - 넓은 공간에서 저속으로 시작
   - 비상 정지 버튼 항상 준비

3. **전기 안전**
   - 작업 전 배터리 분리
   - 쇼트 방지
   - 고전류 주의

### 비상 정지 방법

```
1. 송신기 전원 OFF
2. 키보드 Space 또는 비상 정지 버튼
3. 배터리 분리 (최후 수단)
```
