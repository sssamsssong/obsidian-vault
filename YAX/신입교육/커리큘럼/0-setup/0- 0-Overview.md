# Phase 0: 환경 설정 개요

> YAX F1TENTH 교육자료 | Phase 0: Setup
> 
> 기간: 1-2주차

---

## 1. Phase 0 소개

### 학습 목표
1. Jetson Orin Nano에 JetPack 6.2.2를 설치하고 부팅할 수 있다
2. ROS2 Humble을 설치하고 기본 명령어를 실행할 수 있다
3. 개발 환경(VS Code Remote SSH)을 구축하고 효율적으로 작업할 수 있다
4. MORAI 시뮬레이터를 설치하고 ROS2와 연동할 수 있다

### 전체 소요 시간
- 오프라인 세션: 4시간 (2주 × 2시간)
- 자율학습: 4시간
- **총 8시간**

### 왜 환경 설정이 중요한가?
F1TENTH 프로젝트의 성공은 안정적인 개발 환경에서 시작됩니다. 이 단계에서 환경을 제대로 구축하지 않으면 이후 모든 단계에서 불필요한 시간을 낭비하게 됩니다.

## 2. 하드웨어 요구사항

### 팀 공용 장비
| 장비 | 수량 | 용도 |
|------|------|------|
| Jetson Orin Nano 8GB | 2대 (팀당 1대) | 차량 탑재 컴퓨터 |

| Hokuyo UST-10LX LiDAR | 2대 | 거리 측정 |

| VESC 6+ | 2대 | 모터 제어 |

### 개인 장비
| 장비 | 사양 | 용도 |
|------|------|------|
| 노트북 | Windows 10/11, 8GB+ RAM | 개발, 시뮬레이션 |
| USB 키보드/마우스 | - | Jetson 초기 설정 |
| microSD 카드 | 64GB+, UHS-I | Jetson OS |
| SD 카드 리더기 | - | OS 플래싱 |
